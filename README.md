SpeedBeam ⚡

A modern internet speed test application built with a fast, interactive interface and real-time speed testing.

Features

- ⚡ Real-time internet speed testing
- 📊 Download and upload speed measurement
- 📡 Ping and latency testing
- 🎯 Interactive speedometer interface
- 📈 Speed test results
- 🕘 Test history
- 🤖 AI assistant integration
- 🎨 Modern animated interface
- 📱 Responsive design for desktop and mobile devices

Deployment

SpeedBeam is deployed using Vercel.

The intended workflow is:

Code changes
     ↓
GitHub
     ↓
Vercel
     ↓
Live website

Every update should be committed to the GitHub repository before being deployed to production.

Development

To run SpeedBeam locally:

1. Clone the repository.
2. Open the project folder.
3. Open "index.html" in a browser.

For features that require the API, run the project through a suitable local development server.
